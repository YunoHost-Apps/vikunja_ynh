## Aurreikuspena

Vikunja is a self-hosted open-source to-do list application for all platforms.

### Features

- Stay organized 
- Collaborate with peers
- Tasks  
- Kanban board
- CalDAV
- Links  

**Paketatutako bertsioa:** 0.24.2~ynh2

**Demoa:** <https://try.vikunja.io/login>
