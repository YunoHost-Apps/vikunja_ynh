## Ringkasan

Vikunja is a self-hosted open-source to-do list application for all platforms.

### Features

- Stay organized 
- Collaborate with peers
- Tasks  
- Kanban board
- CalDAV
- Links  

**Versi terkirim:** 0.24.1~ynh1

**Demo:** <https://try.vikunja.io/login>
